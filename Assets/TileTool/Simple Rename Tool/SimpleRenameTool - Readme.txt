Simple Rename Tool is a easy to use game object rename tool for use in scene hierarchy.

FEATURES

        • Replace entire name in  all selected game objects
        • Rename parts of game object names
        • Numerate selected game objects
        • Remove characters from start or end of name
        • Add prefix or suffix to name
        • Save Prefabs - Creates prefabs from selected game objects
                
The tool is located in "Window/Simple Rename Tool"
